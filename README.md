## 🚀 遊び方
1. STARTボタンを押す
2. 隣のブロックをスワイプ
3. 3つ以上揃えると消える
4. 4つ・5つ → 一気にスコア伸びる
5. 💥 コンボ効果 → スコア倍率UP
6. スコアシステム　【スコア = 消した数 × 10 × コンボ倍率】
