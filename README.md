# 🍬 Match3 Ultimate

ブラウザで遊べるマッチ3パズルゲーム

## 🎮 機能
- スワイプ操作（PC / スマホ対応）
- コンボシステム
- 爆発エフェクト
- ハイスコア保存
- スキン切替

## 🚀 遊び方
1. STARTボタンを押す
2. 隣のブロックをスワイプ
3. 3つ揃えると消える
